# 🏪 洛克王国·远行商人 微信推送机器人

基于 GitHub Actions + Server酱 的免费定时推送方案，消息直达微信。

## 📋 功能

- ✅ 每天 8:00 / 12:00 / 16:00 / 20:00（北京时间）自动推送
- ✅ 实时获取远行商人当前在售商品
- ✅ 贵重商品 (≥10w洛克贝) ⭐ 高亮标记
- ✅ 完全免费，无需服务器，无需一直开机
- ✅ 支持手动触发测试

## 🚀 5 分钟部署指南

### 第一步：注册 Server酱

1. 打开 https://sct.ftqq.com/
2. 点击「微信扫码登录」
3. 关注弹出的公众号（方糖）完成激活
4. 复制你的 **SendKey**（在「Key」一栏）

### 第二步：Fork 本仓库

点击右上角 **Fork** 按钮

### 第三步：配置 Secrets

进入你 fork 的仓库 → **Settings** → **Secrets and variables** → **Actions** → **New repository secret**

| Secret 名称 | 值 |
|-------------|-----|
| `SERVERCHAN_KEY` | 上面复制的 Server酱 SendKey |

### 第四步：测试

进入仓库 → **Actions** → **洛克王国·远行商人推送** → **Run workflow** → 点击绿色按钮

几秒后微信会收到消息 🎉
